I Функционал: Работа поиска на сайте https://yandex.ru

ТК001, "URL в результате поиска соответствует запросу"
Шаги:
1. Открыть страницу https://yandex.ru
2. Ввести поисковый запрос "Аргус телеком спб" и нажать кнопку поиска
3. Убедиться, что произошел переход на страницу с результатами поиска
4. ОР: ссылка на сайт argustelecom.ru находится на 1 позиции

ТК002, "Пустой результат пустого поискового запроса"
Шаги:
1. Открыть страницу https://yandex.ru
2. Ничего не вводя в поле поиска нажать кнопку поиска
3. Убедиться, что произошел переход на страницу с результатами поиска
4. ОР: на странице сообщение "Задан пустой поисковый запрос"

II Перечень страниц и методов, участвующих в сценарии:

1. Главная страница - MainPage:
inputSearch - ввод поискового запроса
clickSearch - нажать на кнопку поиска

2. Страница с результатами поиска - SearchPage:
getUrlByNumber - получить URL из списка результатов поиска по номеру позиции
isEmptyResult - проверка, что результат поиска пуст

III Заготовки автотестов

Тесты реализованы по методологии BDD. С использованием Java/Selenium2/Cucumber.
Шаги автотестов описаны в Jerkin нотации.
Возникали проблемы с подбором совместимых и работоспособных версий selenium, драйвера и браузера.
Работоспособность тестировалась на:
Firefox 48.0
selenium-java 3.0.0
geckodriver 0.18.0

Тесты можно запустить с помощью класса my.task.runner.RunTests
Сами тесты находятся в файле resources/features/auth.feature

  @001
  Сценарий: URL в результате поиска соответствует запросу
    Дано Страница ГЛАВНАЯ
    Тогда Заголовок страницы должен быть "Яндекс — поиск №1 в России"
    Если Ввести "Аргус телеком спб" в поле поиска на странице ГЛАВНАЯ
    Если Нажать на кнопку НАЙТИ на странице ГЛАВНАЯ
    Тогда Заголовок страницы должен содержать "Аргус телеком спб"
    Дано Страница РЕЗУЛЬТАТЫ ПОИСКА
    Тогда URL в результате поиска на позиции 1 должен быть равен "argustelecom.ru"

  @002
  Сценарий: Пустой результат пустого поискового запроса
    Дано Страница ГЛАВНАЯ
    Тогда Заголовок страницы должен быть "Яндекс — поиск №1 в России"
    Если Ввести "" в поле поиска на странице ГЛАВНАЯ
    Если Нажать на кнопку НАЙТИ на странице ГЛАВНАЯ
    Тогда Заголовок страницы должен содержать "ничего не найдено"
    Дано Страница РЕЗУЛЬТАТЫ ПОИСКА
    Тогда Результат запроса пуст
